============================================
Float Words（漂浮文字桌面伴侣）
使用说明
============================================

这是一个桌面小工具，会在屏幕上显示温和的漂浮文字提醒。

【快速开始】

1. 双击运行 FloatWords.exe
2. 在系统托盘（右下角）找到程序图标
3. 右键点击图标，选择"开始"
4. 屏幕上会开始出现漂浮文字

【基本操作】

- 开始：开始显示漂浮文字
- 暂停：暂停显示（已显示的文字会继续存在）
- 仅空闲时显示：只在电脑空闲时显示（避免打扰）
- 设置...：打开设置窗口，配置各种选项
- 退出：关闭程序

【设置说明】

1. AI 文本（可选）
   - 如果启用，程序会使用 DeepSeek AI 生成每日不同的文字
   - 需要配置 DeepSeek API Key（在设置窗口中输入）
   - 首次使用需要联网生成，之后会缓存到本地

2. 文本来源
   - auto：自动选择（优先 AI，失败时使用本地）
   - local：仅使用本地文本（data/texts.txt）
   - ai：仅使用 AI 文本

3. 漂浮数量
   - 超多：30 个
   - 多：15 个
   - 普通：8 个（默认）
   - 少：3 个

4. 漂浮速度
   - 快：25ms
   - 正常：40ms（默认）
   - 慢：60ms

5. 称呼和自定义提示词
   - 称呼：AI 生成文字时使用的称呼（可选）
   - 自定义提示词：描述你想要的文字风格（可选）

【数据文件】

程序会在同目录下创建以下文件/目录：

- data/texts.txt：本地文本文件（可以自定义）
- data/ai_cache/：AI 生成的文本缓存（自动管理）

【常见问题】

Q: 为什么没有看到托盘图标？
A: 检查系统托盘区域（右下角），可能需要点击"显示隐藏的图标"。

Q: 如何配置 DeepSeek API Key？
A: 右键托盘图标 → 设置 → 在"DeepSeek API Key"中输入你的 Key。

Q: 可以自定义文字内容吗？
A: 可以！编辑同目录下的 data/texts.txt 文件，每行一条文字。

Q: 程序占用资源多吗？
A: 非常轻量，内存占用通常 < 50MB。

Q: 如何完全退出程序？
A: 右键托盘图标 → 退出，或关闭控制台窗口。

【技术支持】

如有问题，请查看控制台窗口的日志信息。
